## ゲームルール  
タブレット本体を振って炭酸の力を溜め、ジャイロで実際に狙って、ゴミ箱へ炭酸飲料を飛ばすゲーム。
三種類の炭酸飲料や俯瞰視点などを駆使してより少ない打数でゴールを目指す。
